# 古琴数据集

古琴数据集是一个包含古琴曲的符号化音乐数据集。古琴数据集中的古琴谱由我们收集的近年出版的带有简谱的琴谱转录而成。转录的内容为简谱中的旋律与泛音标记，忽略了其他表情记号。

## 制作

我们使用了一种特殊的快速录入文本标记以录入琴谱，录入一页的时间可缩短至3-5分钟。我们之后使用程序将古琴曲转换至MusicXML格式，并自动生成了拍号。
